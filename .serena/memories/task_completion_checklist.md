# Task Completion Checklist (Quality Gate)

変更をアーカイブ(完了)とする前に、以下すべてを満たすこと:

1. [ ] Unity Editor でコンパイルエラーなし
2. [ ] 既存機能への breaking change なし
3. [ ] ASSEMBLY_ARCHITECTURE.md が最新状態
4. [ ] 同名型の重複なし
5. [ ] 循環参照なし
6. [ ] テスト通過 (該当する場合)
7. [ ] 作業レポート作成済み (docs/inbox/ or docs/04_reports/)
8. [ ] Conventional Commits 形式でコミット
