## Unity Version Control（Plastic SCM）への移行を推奨

### GitHub の問題点
- LFS制限（1GB/月）で3Dアセット管理が困難
- .metaファイルの競合が頻発
- シーンファイルのマージが実質不可能

### Unity Version Control の利点
- Unity統合でエディタ内完結
- 大容量ファイル制限なし
- Unity特有のファイル形式に最適化
- 3人まで無料
